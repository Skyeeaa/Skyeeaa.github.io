docker 常用命令

强制删除

从docker中复制文件

如果需要拿docker中的文件

先执行下列命令，更换其当前执行的指令，然后就可以复制

复制文件

docker cp $容器名字:$容器内文件位置 $本机位置

挂载命令-v

外有 覆盖里面，外无 把容器里的挂到外面，永远不会覆盖物理机的文件，挂载多个文件就用多个-v

如果镜像内和本机上都没有这个文件，那么会挂一个目录出来

挂载应当先touch $filename一个出来挂载，本地无文件挂载适用于镜像中已经存在要挂的文件
